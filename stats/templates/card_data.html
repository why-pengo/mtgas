About Card Data
What is this?

MTG Arena logs reference cards by numeric IDs (grpId). To display card names, we use Scryfall's bulk card data, which maps these IDs to card information.

How often should I update?

Update when new Magic sets are released to ensure new cards are recognized. Otherwise, updating every few months is sufficient.

What happens if I don't download?

Without card data, imports will fail or cards will show as "Unknown (grpId)". First-time setup requires downloading this data.

Command Line Alternative

You can also download via command line:
python manage.py download_cards or make download-cards
